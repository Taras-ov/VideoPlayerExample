За основу архитектуры взял классический MVVM, упростил его, так как UseCases посчитал избыточными для такого маленького проекта.

В рамках работы с плеером, сам ExoPlayer закрывает все потребности из под коробки (изменение скорости воспроизведения, перемотка, изменение качества), но так как предоставленный API отдает просто .mp4, а не несколько вариантов одного видео в разных качествах, в плеере смена качества сейчас недоступна. Сделал сначала кастомную заглушку, понял, что нет смысла и убрал. 

Добавил fullscreen режим для приличия:)

Также засетапил Hilt для удобства, хотя для пары экранов можно было бы обойтись и без него.

Ну и весомую часть времени потратил, чтобы проект вообще завелся на agp 9+, версию специально не стал понижать, интересно было прочуствовать, что тут нового, но в подробности вдаваться не буду:)
